\# CoinBank



Projeto desenvolvido em Java que simula um cofrinho digital, permitindo adicionar moedas de diferentes países, listar os valores armazenados e calcular o total convertido para Real (BRL).



\## Descrição do Projeto



O CoinBank foi criado com o objetivo de praticar conceitos fundamentais de Programação Orientada a Objetos (POO), como herança, polimorfismo, encapsulamento e organização de código em múltiplas classes.



O sistema permite ao usuário interagir por meio de um menu no terminal, adicionando moedas e visualizando o valor total acumulado.



\## Funcionalidades



\- Adicionar moedas de diferentes tipos (Real, Dólar e Euro)

\- Listar todas as moedas adicionadas ao cofrinho

\- Calcular o valor total convertido para Real

\- Estrutura orientada a objetos com classes abstratas e especializações



\## Tecnologias Utilizadas



\- Java

\- Eclipse IDE
